productosDisponibles top-level property

List<Producto> productosDisponibles
final

Lista de productos disponibles Cada producto tiene una imagen, un nombre y un precio La imagen es una ruta relativa a la carpeta assets/img El nombre es una cadena de texto El precio es un número decimal Esta lista se utiliza para mostrar los productos en la aplicación
